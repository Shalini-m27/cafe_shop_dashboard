# ☕ Cafe Shop Sales Dashboard

## 📌 Project Overview

The **Cafe Shop Sales Dashboard** project was created using **Microsoft Excel** to analyze and visualize cafe sales data. This project focuses on transforming raw data into meaningful business insights through data cleaning, data processing, data visualization, and interactive dashboard creation.

The dashboard helps businesses understand sales performance, customer behavior, product trends, and store performance to support better decision-making.

---

## 🎯 Project Objectives

* Analyze overall sales performance.
* Track customer footfall and order trends.
* Identify top-performing products.
* Compare store location performance.
* Monitor sales by day, month, and hour.
* Create an interactive dashboard for business insights.
